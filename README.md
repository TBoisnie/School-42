# 42 - AI - Matrix

- Ex01 : Linear combination
- Ex02 : linear interpolation

- Ex04 : Norm
- Ex05 : Cosine

- Ex08 : Trace
- Ex09 : Transpose
- Ex10 : Reduced row-echellon form
- Ex11 : Determinant
- Ex12 : Inverse
- Ex13 : Rank
- Ex14 : Projection matrix
- Ex15 : Complex vector space
